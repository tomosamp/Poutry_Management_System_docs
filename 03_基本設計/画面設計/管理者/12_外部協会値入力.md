# 外部協会値入力画面

## 1. 画面概要
- 目的: ブロイラー全国の外部協会集計値（R◯年全国）を年度単位で登録する。
- 主利用者: 食鳥協会職員。
- 画面カテゴリ: データ提出。

## 2. 主な機能
1. 対象年度の選択（過去年含む）。
2. 対象年度の和暦（令和）を補助表示。
3. ひな入雛羽数/生体処理羽数/生体処理重量の月次値入力（1〜12月）。
4. 保存と更新差分の版管理。

## 3. 画面要素
| ID | セクション | 主な項目 |
| --- | --- | --- |
| SEC-EXT-01 | 対象年度選択 | 対象年度（YYYY）、年度反映ボタン、和暦（令和）補助表示 |
| SEC-EXT-02 | 入力フォーム | ひな入雛羽数（千羽）/生体処理羽数（千羽）/生体処理重量（トン）の月次入力（1〜12月） |
| BTN-EXT-01 | 保存 | 外部協会値を登録 |

## 4. 入力・検証
- 入力単位は「千羽/トン」。保存時に内部単位（羽・kg）へ `×1000` 変換する。
- 12ヶ月分を必須入力（整数・0以上）。
- 対象年度は任意（過年度も入力可）。
- 登録済みの月次値を更新した場合は版番号を+1して保存する。

## 5. 備考
- 保存先は `monthly_aggregate_values`（`data_source=external` / 全国のみ）。
- ALICブロイラー出力時に年号行へ差し込む。
