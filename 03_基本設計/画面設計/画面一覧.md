# 画面一覧

| 画面カテゴリ | 画面名 | 主な目的・機能 | 備考 |
| --- | --- | --- | --- |
| 認証 | ログイン | 共通管理者アカウントで認証し、管理者のみが各機能へアクセスできるようにする。 | パスワード変更導線は設置しない。 |
| ダッシュボード | 管理ダッシュボード | 当月の未提出生産者と提出件数を俯瞰し、優先対応を明確化する。 | 未提出一覧（提出履歴一覧の未提出フィルタ）への導線を提供。 |
| 生産者管理 | 生産者一覧 | 生産者名検索、鳥種・地域ブロックフィルタ、最終提出日表示で生産者を横断管理する。 | ステータス別バッジ表示を行う。 |
| 生産者管理 | 生産者詳細・編集 | 生産者の基本情報、所在地、代表者/申請者情報、特記事項メモを閲覧・更新する。 |  |
| 生産者管理 | 生産者新規登録 | 生産者情報と申請者情報を登録し、都道府県に応じた地域ブロックを設定する。 | ブロックは自動初期値 + 手動調整可。 |
| データ提出 | CSVアップロード | 鳥種別×提出月別の一括CSVを取り込み、検証後に入力確認へ進める。 | 検証結果からOK行のみ確認画面へ進める。 |
| データ提出 | 手入力登録 | 生産者別フォームで鳥種ごとの項目を入力し、コメントも含めて登録する。 | 入力中のリアルタイム検証を行う。 |
| データ提出 | 入力確認 | CSVおよび手入力から遷移し、数値修正や警告確認を経て提出を確定する。 | 代理提出者と対象月を記録。 |
| データ提出 | 提出履歴一覧 | 対象月の提出済/未提出を一覧化し、詳細確認や再提出へ繋げる。 | 未提出抽出は本画面で行う。 |
| データ提出 | 提出履歴詳細 | 提出日時、担当職員、項目ごとの修正履歴を時系列で確認する。 | 再提出操作および差分表示を提供。 |
| 集計・出力 | 月次/年次レポート | 提出先を選択し、指定フォーマットに整形してExcel/PDFで出力する。 | 出力履歴（再出力）を提供。 |
| 共通UI | 99_サイドバー | 全画面共通のグローバルナビゲーションを提供し、主要機能への導線を一元化する。 | 上部: ロゴ（02）。セクションA: 生産者管理（03）。セクションB: 集計・出力（11）。ドロップダウン: データ提出（06/07/09）。下部: ログアウト（01）。 |
