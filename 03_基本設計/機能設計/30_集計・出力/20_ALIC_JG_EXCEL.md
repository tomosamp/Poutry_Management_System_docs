# ALIC_JG_EXCEL 提出フォーマット仕様

## 1. 対象
- 提出先: エーリック（農畜産業振興機構）
- 鳥種: 地鶏
- 集計: 全国
- 形式: Excel

## 2. テンプレ配置
- 配置先: `app/src/storage/app/export_templates/ALIC_JG_EXCEL.xlsx`
- テンプレコードとファイル名は `export_templates.code` で紐付ける。

## 3. 出力ファイル名
- 形式: `地鶏（ALIC）｜R{令和年}年{提出月}月地鶏機構報告.xlsx`
- `{提出月}` は **提出月（締切月）** とする。
- 令和年は `{提出月}` の西暦から算出する（`西暦 - 2018`）。
- 例:
  - 提出月が2026年2月の場合 → `地鶏（ALIC）｜R8年2月地鶏機構報告.xlsx`
  - 提出月が2026年1月の場合 → `地鶏（ALIC）｜R8年1月地鶏機構報告.xlsx`

## 3.1 シート名
- 形式: `R{令和年}年{実績月}月実績報告`
- `{実績月}` は **提出月の先月** とする。
- 例:
  - 提出月が2026年2月の場合: `R8年1月実績報告`
  - 提出月が2026年1月の場合: `R7年12月実績報告`

## 4. セルマッピング（基本構成）
### 4.0 年度表記セル
- タイトル/作成日
  - `B4`: `地鶏生産・処理動向調査報告書（令和{R}年度）`
    - `{R}` は **対象年度（4月開始の年度）** の西暦から算出する（`西暦 - 2018`）。
  - `Z2`: 出力日（`YYYY/MM/DD`）
- 今年度の年度表記（文字列 `令和{R}年度`）
  - `B10`, `B19`, `B29`
  - `{R}` は **対象年度（4月開始の年度）** の西暦から算出する（`西暦 - 2018`）。
- 前年度の年度表記（文字列 `令和{R-1}年度`）
  - `B11`, `B20`, `B34`
  - `{R-1}` は上記 `{R}` の1つ前（対象年度-1）とする。

### 4.1 月次列（4月〜3月）
- 4月〜3月のレイアウトで、セル結合の関係により列は1つ飛ばしになっている。
- 月次列:
  - 4月: `C`
  - 5月: `E`
  - 6月: `G`
  - 7月: `I`
  - 8月: `K`
  - 9月: `M`
  - 10月: `O`
  - 11月: `Q`
  - 12月: `S`
  - 1月: `U`
  - 2月: `W`
  - 3月: `Y`
- 年度合計: `AA`（全ての月の合計）

### 4.2 行と項目
- 出力単位
  - 羽数系（生体処理羽数/禁止/全部廃棄/一部廃棄）は **千羽単位**（`値/1000` を四捨五入）
  - 重量（生体処理重量）は **トン単位**（`kg/1000` を四捨五入）
- 生体処理羽数
  - 今年度: `10` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
  - 前年度: `11` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
- 生体処理重量
  - 今年度: `19` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
  - 前年度: `20` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
- 禁止/廃棄
  - 禁止
    - 今年度: `31` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
    - 前年度: `36` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
  - 全部廃棄
    - 今年度: `32` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
    - 前年度: `37` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
  - 一部廃棄
    - 今年度: `33` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
    - 前年度: `38` 行（各月: `C,E,...,Y` / 年度合計: `AA`）
- 集計した生産者の会社数（合計）
  - `13` 行（羽数セクション）
  - `22` 行（重量セクション）
  - `30` 行（処分セクション・今年度）
  - `35` 行（処分セクション・前年度）
  - 各月: `C,E,...,Y` / 年度合計: `AA`

### 4.3 背景色（実績月ハイライト）
- 毎月の出力で対象となる **実績月（= 提出月の先月）** の列は、背景を黄色でハイライトする。
- 対象列は、実績月（4月〜3月）に対応する列（`4.1 月次列` に従う）。
  - 基本: `C,E,...,Y` の該当列を着色する（セル結合により1列飛ばしのため）。
  - **29〜33行目** はセル結合がないため、月の2列分を着色する（例: 4月は `C:D`、5月は `E:F` … 3月は `Y:Z`）。
- ハイライト対象行（該当列のみを着色）:
  - `9〜13` 行
  - `18〜22` 行
  - `28〜33` 行
